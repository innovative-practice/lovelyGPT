# 项目进度

|  接口   | 描述  |
|  ----  | ----  |
| List Models  | 获取模型列表 |
| Chat Completion  | GPT3.5模型 |
| Completion  | GPT3.5以下模型 |
| Create Image  | 根据描述生成图片 |
| Create Image Variation  | 根据上传的图片生成变体图片 |
| Create transcription  | 音频识别文字 |
| Create translation    | 英语音频识别 |
| fine-tune    | 待开发 |
| Files    | 待开发 |
